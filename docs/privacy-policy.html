Redox Launcher Privacy Policy

Last updated: July 22, 2026

Overview

Redox Launcher is designed to keep launcher data private by default. Crash reports and usage analytics are off until you turn them on in Security and privacy.

Data stored on this device

Launcher preferences, hidden app choices, home layout, search history, PIN state, and recovery phrase verification data are stored locally on your device using app storage.

If you create a backup, Redox saves selected launcher preferences and layout data to a backup file. Manual backups are saved to the location you choose. If automatic backup is enabled, Redox saves backup files locally in Downloads/RedoxLauncher/Backups and keeps only the number of latest backups you choose.

Optional feature data

Some features use Android permissions only when you enable or use them:

  • Contacts search reads contacts locally for search results.
  • Weather may use approximate location to request local weather.
  • Now Playing may use notification access to show active media and music/audio access to reopen remembered local audio.
  • Smart controls may use Bluetooth, camera, and Do Not Disturb access for local device controls.
  • Accessibility access may be used for launcher gesture or automation features that you enable. Redox does not use Accessibility access to read passwords, payment information, or unrelated screen content.
  • Cloud backup upload sends the backup file only to the cloud app or provider you select, such as OneDrive, Google Drive, or Dropbox.

Optional diagnostics

If Crash reports is enabled, Google Firebase Crashlytics may receive crash details such as app version, device model, Android version, crash time, stack traces, and diagnostic logs to help fix stability issues.

If Usage analytics is enabled, Google Firebase Analytics may receive generic launcher events such as settings_opened, app_drawer_opened, search_opened, gesture_used, widget_added, folder_created, and wallpaper_changed.

Redox does not request the Android Advertising ID and does not use analytics for advertising, ad targeting, or selling user data.

Data sharing

Redox does not sell personal data. Optional crash reports and usage analytics may be processed by Google Firebase only when those features are enabled by the user.

Data not sent by Redox analytics

Redox analytics does not send personal app names, package names, contact names, search text, hidden app lists, PINs, recovery phrases, files, photos, precise location, or home screen contents.

Your controls

You can turn Crash reports and Usage analytics on or off at any time from Security and privacy. Turning a toggle off immediately stops that Firebase collection path in the app. You can also disable automatic backup at any time from Backup and restore.

Retention and deletion

Local launcher data remains on your device until you change settings, use Delete local privacy data, clear app data, or uninstall Redox. Optional Firebase diagnostic and analytics data is retained according to the project retention settings for those Google services.

Backup files remain where they are saved until you delete them. Automatic backup deletes older automatic backup files based on the Keep latest backups setting.

Children

Redox Launcher is not directed to children and is intended for general users.

Security

Redox uses Android app storage and platform permission controls to help protect local launcher data. No method of storage or transmission is completely secure.

Privacy contact
